# 💰 Expense Tracker API

A simple Expense Tracker backend built using **FastAPI** and **SQLAlchemy**.

---

## 🚀 Features
- Add expenses
- View all expenses
- Update expenses
- Delete expenses
- Category-wise tracking

---

## 🛠️ Tech Stack
- Python
- FastAPI
- SQLAlchemy
- SQLite (or your DB)

---

## ▶️ How to Run

```bash
# Activate virtual environment
venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt

# Run server
uvicorn main:app --reload
